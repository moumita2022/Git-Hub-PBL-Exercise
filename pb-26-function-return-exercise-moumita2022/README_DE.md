    Zum Beispiel:
    ```js
        min(2, 5) == 2
        min(3, -1) == -1
        min(1, 1) == 1
    ```

    ```js
    pow(3, 2) = 3 * 3 = 9
    pow(3, 3) = 3 * 3 * 3 = 27
    ```
